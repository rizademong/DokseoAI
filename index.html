프로젝트 상세

문제 정의

책에 대한 감정적이고 설득력 있는 리뷰를 자동으로 생성하여 사용자에게 구매 욕구를 유도하는 AI 리뷰 시스템을 목표로 합니다.

아키텍처 및 설계

파이썬 모델을 기반으로 리뷰 데이터를 학습하고 자연어 생성 기능을 구현했습니다.

핵심 기능 1: 온라인 및 제한적 오프라인 사용

온라인 학습 데이터를 로컬에 저장하여 오프라인 상황에서도 기존 데이터 기반 리뷰 생성을 지원합니다.

핵심 기능 2: 데이터 초기화 및 AI 수정

잘못된 학습 데이터나 성능 저하 발생 시 내부 데이터를 초기화하여 안정적인 상태로 복원할 수 있습니다.

핵심 기능 3:

챗봇의 모드를 도슨트와 일반 리뷰로 분류해 사용자 취향의 리뷰 데이터를 설명합니다.

나의 역할

전체 설계, 개발, 테스트를 수행했습니다.

결과 및 성과

현재 개발 진행 중이며 Gemini API를 통한 초기 리뷰 생성 기능이 작동 중입니다.

회고

  • 배운 점: AI 모델 활용을 위해 구조적 이해와 데이터 처리 능력의 중요성을 체감했습니다.
  • 개선점: 경량화 모델인 Gemma2:b의 단기 기억상실 현상을 조절해야 합니다.
  • 개선점: 배포 단계와 QA단계까지 수행해야 합니다.
