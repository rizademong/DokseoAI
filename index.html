🧩 문제 정의

책에 대한 감정적이고 설득력 있는 리뷰를 자동으로 생성해, 사용자에게 구매 욕구를 불러일으키는 AI 리뷰 시스템을 구현했습니다.

🏗️ 아키텍처 / 설계

Gemini 모델을 기반으로 리뷰 데이터 학습 및 자연어 생성 기능을 구현했습니다.

⚙️ 핵심 기능 1: 온라인 사용 가능 및 제한적 오프라인 기능

온라인으로 학습한 데이터를 로컬에 저장하여, 오프라인일 경우에도 저장된 데이터에 기반한 리뷰 작성이 가능합니다.

⚙️ 핵심 기능 2: 데이터 삭제 및 리뷰 AI 수정

AI가 잘못된 데이터를 학습하거나 성능 저하가 감지되면, 내부 학습 데이터를 자동으로 초기화하여 복원할 수 있습니다.
